Introducing Pixelated, a lightweight FSE (Full Site Editing) Gutenberg Block WordPress theme designed for bloggers. This minimal theme offers a clean, modern design focused on blog content, with Montserrat for headings and Roboto for body text. It's optimized for speed and simplicity, leveraging WordPress core features for customization. Perfect for writers, journalists, and anyone seeking a distraction-free blogging experience.

The theme includes Google Fonts: Montserrat (https://fonts.google.com/specimen/Montserrat) and Roboto (https://fonts.google.com/specimen/Roboto), licensed under the Open Font License.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support widgets? =

No, this is a full site editing theme that uses blocks for customization.

= How do I customize the theme? =

Use the WordPress site editor (Appearance > Editor) to modify templates, colors, and layouts.

= Is the theme responsive? =

Yes, it uses fluid typography and responsive layouts.
